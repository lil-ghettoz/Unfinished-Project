# Z Shield

## Project Description
Create a modern, high-end mobile application called **“Z Shield”**, a personal safety and cybersecurity app designed to protect users in both real-life emergencies and digital threats.

The app should have a dark, futuristic UI theme (black, deep blue, neon accents) that feels like a professional security system. The overall experience must feel serious, responsive, and trustworthy.

---

## 🎯 CORE PURPOSE

Z Shield acts as a **personal bodyguard + digital protection system**. It allows users to instantly alert trusted contacts, record evidence during emergencies, and stay protected from scams and unsafe situations.

---

## 🔥 MAIN FEATURES

### 1. 🚨 Panic Button (Primary Feature)

* A large, centered red panic button on the home screen
* When pressed:

  * Automatically captures the user’s GPS location
  * Starts recording video and audio immediately
  * Sends emergency alerts via SMS and push notifications to selected contacts
  * Generates a live tracking link that can be shared
  * Uploads recorded data to secure cloud storage

---

### 2. 🎥 Auto Evidence Recording

* Automatically records video/audio during emergencies
* Files are securely uploaded to the cloud in real time
* Even if the device is lost or destroyed, evidence remains accessible

---

### 3. 📍 Live Location Tracking

* Real-time GPS tracking with a shareable private link
* Displays movement on a map
* Allows trusted contacts to monitor user location

---

### 4. 👥 Emergency Contacts Management

* Users can add trusted contacts
* Toggle who receives alerts
* Import contacts from phone

---

### 5. 🔐 Security & Privacy Features

* Panic PIN (required to cancel emergency mode)
* Optional fake unlock mode (disguised app interface)
* Encrypted data storage
* Permission control dashboard

---

### 6. ⚠️ Scam & Threat Detection (Phase 2 Feature)

* Detect suspicious links or messages
* AI assistant that answers: “Is this a scam?”
* Warn users about risky apps or permissions

---

## 🎨 UI / UX DESIGN REQUIREMENTS

* Dark mode interface (primary color: #0B0F1A)
* Neon accents (red for danger, blue for safety)
* Smooth animations and glowing effects
* Minimal but powerful layout

### Key Screens:

1. Splash Screen (logo + tagline)
2. Login / Signup
3. Home Screen (with large panic button)
4. Panic Mode Screen (recording + alerts active)
5. Contacts Screen
6. Live Tracking Screen (map view)
7. Settings
8. Alert History
